# Ecommerce Microservice

This is a NestJS microservice that provides product management, user management, authentication, authorization, logging, caching functionality.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Features

- User Management
- Product Management
- Authentication
- Authorization
- Logging
- Caching
- Database
